# SpeakThatSpanish PWA Static Files

Upload everything in this folder to the root of your GitHub Pages repository.

Required root files:
- index.html
- manifest.webmanifest
- service-worker.js
- favicon.ico
- icons/ folder

GitHub Pages:
1. Put these files at the repository root.
2. Go to Settings > Pages.
3. Source: Deploy from branch.
4. Branch: main, folder: /root.
5. Wait 1–2 minutes.

Install behavior:
- Android Chrome should show "Add to Home screen" or "Install app".
- iPhone Safari: Share button > Add to Home Screen.
